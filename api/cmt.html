# web3.cmt

The web3.cmt contains the CyberMiles blockchain related methods. Refer to Ethereum's web3.eth for detailed information.

# Example

// send a transfer transaction
web3.cmt.sendTransaction(
  {
    from: "0xde0B295669a9FD93d5F28D9Ec85E40f4cb697BAe",
    to: "0x11f4d0A3c12e86B4b5F39B213F7E19D048276DAe",
    value: web3.toWei(100, "cmt")
  },
  (err, res) => {
    // ...
  }
)

// get the balance of an address
var balance = web3.cmt.getBalance("0x11f4d0A3c12e86B4b5F39B213F7E19D048276DAe")

# web3.cmt.syncing

web3.cmt.syncing
// or async
web3.cmt.getSyncing(callback(error, result){ ... })

This property is read only and returns the sync object. JSON RPC method: cmt_syncing.

# Returns

  • Object - A sync object as follows:

    • latest_block_hash: Number - The hash of the latest block.
    • latest_app_hash: Number - The hash of latest application state.
    • latest_block_height: Number - The latest block number.
    • latest_block_time: Number - The latest block time.
    • syncing: Boolean - Whether the node is syncing or not.

# Example

var sync = web3.cmt.syncing
console.log(sync)
/*
{
  latest_block_hash: '3F84EF726F19DF528620DD745FC6C2251CAE2673',
  latest_app_hash: '64BA27DCACAB23EBF4CF7ACE3604285F66739226',
  latest_block_height: 635,
  latest_block_time: '2018-06-20T08:01:31.061402192Z',
  syncing: false
}
*/

# web3.cmt.pendingTransactionCount

web3.cmt.pendingTransactionCount
// or async
web3.cmt.getPendingTransactionCount(callback(error, result){ ... })

This property is read only and returns the number of unconfirmed transactions in the mempool. JSON RPC method: cmt_pendingTransactionCount.

# Returns

  • Number - The number of unconfirmed transactions in the mempool.

# Example

var count = web3.cmt.pendingTransactionCount
console.log(count)
/*
1
*/

# web3.cmt.getPendingTransactions

web3.cmt.getPendingTransactions([limit], [callback])

Get unconfirmed transactions in the mempool. JSON RPC method: cmt_getPendingTransactions.

# Parameters

  • limit: Number - (optional) The maximum number of transactions returned in the response body. Default: 30, maximum: 100.
  • callback: Function - (optional) If you pass a callback the HTTP request is made asynchronous. See this note

# Returns

  • Object - An array of unconfirmed transaction objects in the mempool.

# Example

var txs = web3.cmt.getPendingTransactions()
console.log(JSON.stringify(txs, null, 2))
